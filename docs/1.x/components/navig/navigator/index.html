组件属性#

NameTypeDescription
frontColorstring前景颜色值，包括按钮、标题、状态栏的颜色，仅支持 #ffffff 和 #000000
backgroundColorstring背景颜色值，有效值为十六进制颜色
animationObject动画效果
[success]function接口调用成功的回调函数
[fail]function接口调用失败的回调函数
[complete]function接口调用结束的回调函数（调用成功、失败都会执行）
